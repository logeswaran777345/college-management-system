# College Management System

## 📌 Description
This is a Java-based College Management System using JDBC and MySQL.

## 🚀 Features
- Student Management
- Teacher Management
- Marks Management
- GUI using Swing

## 🛠 Technologies
- Java (JDBC, Swing)
- MySQL (Database)

## ⚠️ Note
This project requires MySQL to run locally. Database file (database.sql) is included in this repository.

## ▶️ How to Run
1. Install MySQL
2. Run database.sql
3. Update DB credentials in Java code
4. Run the Java files
